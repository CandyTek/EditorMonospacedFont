# 中英等宽字体
本项目收集 代码编辑器 里的中英（中文与英文）2比1等宽字体。


<br>

> 混合配置，完美等宽
> 
> 打开 Global Options 文件
> 
> font.monospaced.list=xxx;xxx
```
JetBrains Mono;Microsoft YaHei Mono
```
