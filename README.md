# 中英等宽字体
本项目收集 代码编辑器 里的中英（中文与英文）2比1等宽字体。


<br>

### 注：部分字体配置需调整字号

# Sublime Text 3 < Build 3126

> 
> 打开 Preferences.sublime-settings User 文件
> 
> "font_face": "第一字体,第二字体",
>
> "font_options": ["no_bold", "no_italic" ], // 非gdi配置
